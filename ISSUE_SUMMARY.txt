COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════════

Q: How long does startup take?
A: First time: 30-60 seconds (building images)
   Later times: 10-15 seconds (just starting containers)

Q: Do I need to rebuild every time?
A: No, just use: docker-compose up
   Use --build only if you change code

Q: Can I run in background?
A: Yes: docker-compose up -d
   Then check status: docker-compose ps

Q: How do I see if there are errors?
A: docker-compose logs
   Or specific service: docker-compose logs backend
